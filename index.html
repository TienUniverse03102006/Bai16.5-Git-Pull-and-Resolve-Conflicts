Một dự án có nhiều người làm và họ có thể sửa

    Trên máy ông A

  1. Đăng bài lên github và deploy như thường
  2. Trên github vào settings ->collaborators->enter password
  3. Add people

    Trên máy ông B

  1. Vào github, bấm vào ảnh đại diện góc phải màn hình
  2. Đến organizations-> accept
  3. Tạo một thư mục, chưa tạo file vội
  4. Trên Vscode, open folder vừa tạo trên Vscode
  5. Đến terminal, xong gõ git init
  6. git remote add origin 'địa chỉ file của ông A'
  7. git branch -m main
  8. git pull origin main
